Teams
Each team gets a unique start tag programmed just for them. This identifies which team is playing on a device. All other tags are shared.
Trail Tags (in order)
Add tags in the order teams should find them. The last tag = finish line. Teams can skip intermediate tags if they find the finish directly.
Red Herring Tags
Scattered around the house. Scanning one adds 60 seconds to the team's time.
Clues
Text shown on the team's phone after each scan. Leave blank for a default message.
Penalty Duration
Hosted URL
Host this file on GitHub Pages (free) or any web host. Paste your URL here to generate NFC tag links.
Multi-Device Scoring
Each team plays on their own phone — scores are local. When a team finishes, they see a Result Code. The host types this into the Dashboard to record the score and build the leaderboard.
URLs to Write to NFC Tags
Use a free NFC writer app (NFC Tools, TagWriter). Write each URL to its matching sticker. Tapping the sticker opens this page with the right tag loaded.
